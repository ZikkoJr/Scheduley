# Schedule-Builder

A university schedule builder that helps students plan their week by combining:
- Course times
- Work shifts
- Personal goals (study, gym, projects)

The app will automatically generate a structured weekly schedule and provide a calendar view.

---

## Project Goals
- Learn full-stack development (Java Spring Boot, React, PostgreSQL)
- Integrate AI/LLM parsing later for syllabus files
- Build a strong portfolio project for internships

---

## Tech Stack (Planned)
- **Backend:** Java 21 + SQLite
- **Frontend:** JavaFX + Maven
- **Database:** SQLite
- **Version Control:** Git + GitHub
